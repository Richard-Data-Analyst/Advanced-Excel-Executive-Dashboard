# Global Retail Executive Performance Dashboard (Advanced Excel)

## 📌 Project Overview
This project transforms over 9,000 rows of unpolished transactional data into a dynamic, self-service executive reporting canvas built entirely within Microsoft Excel. It automates data cleaning and isolates structural commercial performance variables across products, regions, and consumer cohorts.

## 🛠️ Technical Implementation Stack
* ETL Engine:* Excel Power Query Editor
* Logical Data Layer:* Locale-based schema mapping & type casting
* Analytical Core:* Dynamic Pivot Tables & Integrated Calculated Fields
* User Interface:* Interactive Slicers with multi-table cross-filtering (Report Connections)

## 🚀 Key Technical Milestones
* Regional Date Cleansing:* Overcame critical text-to-date transformation failures caused by UK/US date locale conflicts using targeted Locale-Based Type Ingestion inside Power Query.
* Calculated Field Engineering:* Bypassed unsafe hardcoded worksheet formulas by embedding a dynamic Profit / Sales calculation layer directly into the Pivot engine.
* Multi-Axis Visualization:* Developed a synchronized combo chart utilizing dual-axis properties to visualize high-volume financial revenue alongside micropercentage profit margins.
*   Interface Control Panel:* Configured responsive cross-filtering slicers mapped to backend multidimensional grids to allow instantaneous, zero-code report updates for stakeholders.

## 📈 Commercial Insights Discovered
*   Technology* is driving a massive *18.43% profit margin*, making it the most lucrative segment to scale.
*   *Furniture* is barely breaking even with a dangerously low *2.03% margin*, identifying an immediate operational bottleneck.
*   The *Home Office* consumer segment delivers the highest overall financial efficiency at a *15.53% profit margin*.
